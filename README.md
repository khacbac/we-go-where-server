# we-go-where-server

## Cloud application hosting
Render: https://render.com/

## Running the app

1. Clone the repository:

```
git clone https://github.com/khacbac/we-go-where-server.git
cd we-go-where-server
```

2. Install node_modules in project root directory:

```
yarn
```

### start application

```
yarn start
```
